# Legacy's Predator - Release Roadmap

This directory contains details about the development roadmap and release cycles for **Legacy's Predator**.

## Directory Contents

- [Root Roadmap](../../ROADMAP.md) - Main roadmap and high-level milestones.
- Release Schedules (TBD):
  - **Milestone 0.1.0 (Alpha)**: Foundational single-agent CLI framework, basic file system tool execution, and local database storage.
  - **Milestone 0.2.0 (Beta)**: Multi-agent coordination protocols, semantic memory retrieval via vector databases, and interactive UI web client.
  - **Milestone 1.0.0 (Production)**: Fully sandboxed operating environment, multi-platform connectors, and enterprise-grade security and role-based access.
